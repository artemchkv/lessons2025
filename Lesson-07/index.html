Методы REST API

GET

GET для получения данных. Не содержит бади, только адресная строка

    Ожидаемые ответы
  • 500 - server error
  • 400 - bad requesr
  • 401 - Unauthorized
  • 404 - Not found
  • 403 - Forbidden
  • 302 - Not found
  • 304 - Not Modified
  • 200 - Ok
  • 201 - Created

POST

POST для создания данных. Содержит бади, но и може в адресной строке содержать Query params

PATCH

PATCH для редактирования данных. Содержит бади, но и може в адресной строке содержать Query params

PUT

PUT для редактирования данных. Содержит бади, но и може в адресной строке содержать Query params

DELETE

DELETE для удаление данных.
